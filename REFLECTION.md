# Reflection: Challenges in Balancing Stakeholder Needs

# Key Challenges Faced:
Conflicting Priorities  
   - Project managers want detailed tracking, while team members prefer simpler interfaces.  
   - Solution: Implement customizable dashboards for different roles.  

Security vs Usability  
   - IT Admins demand high security (MFA, encryption), but this adds complexity for users.  
   - Solution: Offer  biometric login for convenience while maintaining security.  

Scalability ConcernS  
   - Executives want **large-scale analytics**, but too much data slows performance.  
   - Solution: Use **indexed databases & optimized queries** for reports.  

4️⃣ **Notifications Overload:**  
   - Users want **reminders**, but too many notifications cause frustration.  
   - Solution: Allow **customizable notification preferences**.  

### **Lessons Learned**
✅ Balancing usability, security, and performance requires trade-offs.  
✅ Engaging stakeholders early **prevents conflicts later** in development.  
✅ Agile methodology **helps adjust priorities** as project needs evolve.
